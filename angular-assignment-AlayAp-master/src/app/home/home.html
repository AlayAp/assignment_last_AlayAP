Welcome to Angular Feedback App

This is a modern Angular 18 application showcasing client-side navigation, API integration, and reactive forms. Built with standalone components and the latest Angular features.

Features

🏠 Home Page

A clean landing page with introductory content about the application.

📊 API Integration

Fetch and display data from external APIs using Angular's HttpClient service.

📝 Reactive Forms

Submit feedback through validated reactive forms with real-time validation.

🧭 Client-Side Routing

Seamless navigation between pages using Angular Router.
